# Slowenisch lernen

Lern-Workspace für Slowenisch, betrieben mit dem [`/teach`](https://www.aihero.dev/learn-anything-with-my-teach-skill)-Skill von Matt Pocock.

## Wie es funktioniert

`/teach` behandelt dieses Verzeichnis als Lern-Workspace. Der Fortschritt liegt in:

- `MISSION.md` — warum ich Slowenisch lerne (Grundlage für alle Lektionen)
- `RESOURCES.md` — geprüfte Quellen
- `lessons/*.html` — einzelne, in sich geschlossene Lektionen
- `reference/*.html` — Spickzettel & Referenzmaterial (z. B. Glossar)
- `learning-records/*.md` — was ich gelernt habe, für die nächste Session
- `NOTES.md` — Lernpräferenzen

## Nutzung

In diesem Ordner Claude Code öffnen und ausführen:

```
/teach Slowenisch
```

Der Skill liegt projektlokal unter `.claude/skills/teach` und wird mit ins Repo committet, damit er auch über die Cloud verfügbar ist.
