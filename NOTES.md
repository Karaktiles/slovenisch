# Notes — Lernpräferenzen

- **Lernsprache / Immersion:** Möglichst viel Slowenisch im Lernmaterial. Slowenische Vokabeln, Sätze und Beispiele bevorzugt auf Slowenisch halten; Erklärungen/Hinweise nur so viel Deutsch wie nötig zum Verständnis. Immersiver, fordernder Ansatz.
- **Muttersprache des Lernenden:** Deutsch (Vorarlberg, Österreich) — aber Slowenisch war die *erste* Sprache als Kind. Siehe Profil in `learning-records/0001`.
- **Profil:** Heritage Speaker. Verstehen stark, Sprechen/Lesen schwach. Auf Produktion trainieren, nicht auf Verständnis. Niveau ruhig fordernd ansetzen.

## Publishing (Handy-Zugang) — STEHENDE REGEL
- Repo ist **öffentlich**, ausgeliefert via **GitHub Pages** von `main` (root). Live: https://karaktiles.github.io/slovenisch/
- **Seb lernt am Handy. Pflicht bei JEDER neuen Lektion (von Seb ausdrücklich gewünscht):**
  1. Lektion in `index.html` verlinken (Lektionen + ggf. Referenz)
  2. `git add -A && commit && push origin main`
  3. Bauen abwarten (~1 Min) und **kurz verifizieren** (curl 200), dass die Lektion live ist
  4. Seb den **direkten Link** zur neuen Lektion + zur Startseite geben
- Niemals eine Lektion nur lokal lassen. „Fertig" heißt: live über den Link erreichbar.

## Variante: Hochslowenisch (knjižna slovenščina)
- **Entscheidung (von der Mutter empfohlen):** Standard-/Hochslowenisch lernen, NICHT den Heimat-Dialekt. Damit ist die frühere Dialekt-Frage erledigt.
- Passt ohnehin: alle Quellen (slonline, Fran, Forvo-Standardaufnahmen) lehren Standard. Für Alltag/Familie zielen wir aufs *knjižni pogovorni* Register (Standard-Umgangssprache), nicht steifes Nachrichten-Slowenisch.
- Lektion 1 bleibt gültig — die gelehrte l→[w]-Regel etc. gehört zum Standard.
